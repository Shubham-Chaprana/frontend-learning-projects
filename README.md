# frontend-learning-projects
A collection of frontend projects built while learning HTML, CSS, JavaScript, and React. This repository documents my hands-on learning journey through mini projects, experiments, games, and interactive web apps.

## Projects

### Passenger Counter App
A simple JavaScript app that tracks the number of passengers entering a location. Users can increment the count and save previous entries. Built to practice:
- DOM manipulation
- Event handling
- Functions
- Updating UI dynamically with JavaScript

 ### Blackjack Game
A simple browser-based Blackjack game built with JavaScript. Built to practice:
- Arrays and objects
- Conditional logic
- Random number generation
- DOM manipulation
- Game state management

 ### Chrome Lead Tracker Extension
A simple Chrome extension built to save and manage useful links and browser tabs. Users can manually save links, store the current tab, and persist data using local storage. Built to practice:
- Chrome Extension APIs
- Local Storage
- DOM manipulation
- Event listeners
- Dynamic rendering with JavaScript

  ### Travel Journal (React)
A travel journal web app built with React that displays travel entries with images, dates, descriptions, and Google Maps links. Built to practice:
- React components
- Props
- Rendering lists with `.map()`
- Importing assets in React
- Dynamic UI rendering
- Component-based project structure

  ## Meme Generator (React)

A meme generator built with React that fetches meme templates from an external API and allows users to create custom memes with personalized text.

**Concepts practiced:**
- React state management (`useState`)
- Side effects with `useEffect`
- API fetching
- Controlled form inputs
- Event handling
- Dynamic rendering

---

## Tenzies Game (React)

A dice game where the goal is to roll until all dice show the same value. Players can hold selected dice between rolls and try to win in the fewest moves possible.

**Concepts practiced:**
- React state management
- Complex state updates
- Conditional rendering
- Array methods (`map`, `every`)
- Component composition
- Game logic implementation

---

## Assembly Endgame (React)

A word-guessing game inspired by Hangman, where players must guess letters correctly before all programming languages are eliminated.

**Concepts practiced:**
- Managing complex application state
- Derived state
- Conditional UI rendering
- Event handling
- Dynamic styling
- Game logic implementation

## Purpose

This repository serves as a record of my frontend development journey, showcasing projects built while learning modern web development concepts through hands-on practice. It includes projects ranging from JavaScript fundamentals and browser extensions to interactive React applications, reflecting my growth in building dynamic and user-friendly web experiences.
